# DeerFlow 面试资料索引

## 文件说明
- `DEERFLOW_STUDY_GUIDE.md`: 总览学习手册，包含项目结构、5 条主线、阅读顺序和 10 天学习路线。
- `DEERFLOW_INTERVIEW_QA.md`: 高频问答卡，适合面试前快速背诵和查漏补缺。
- `DEERFLOW_5_MIN_PITCH.md`: 30 秒、2 分钟、5 分钟三个版本的项目讲稿。

## 推荐使用顺序
1. 先看 `DEERFLOW_STUDY_GUIDE.md`，建立全局图。
2. 再看 `DEERFLOW_INTERVIEW_QA.md`，练习高频题。
3. 最后看 `DEERFLOW_5_MIN_PITCH.md`，做口头表达训练。
